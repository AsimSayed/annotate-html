Chrome Extension · v1.0

Annotate any page1
on the web.

Click an element on any site — your own build, a staging URL, a competitor, a design reference — type what you want changed, and paste structured markdown into Claude Code.2

Add to Chrome Source

Also embeddable as a <script> tag for HTML pages you build.3

1 Works everywhere

Any page you can visit: staging environments, production sites, competitor pages, Figma prototypes, shared design references. No source access needed — the extension injects the toolbar on demand.

2 What it captures

Each annotation records element name, full selector path, CSS classes, bounding box, computed styles, nearby text, and accessibility data. Four verbosity levels — compact, standard, detailed, forensic — so the output matches what your agent actually needs.

3 Zero dependencies

24 KB. ES5 IIFE. No build step, no npm, no framework. The Chrome extension and the <script> tag share the exact same file — one codebase, two distribution channels.
