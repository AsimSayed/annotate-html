Workflow

Four clicks from pixel to PR.

You've already used the pattern — the loop between seeing something and saying "make that softer" is what this shortens.

Step one

Drop the script

One line before </body>. No build, no bundler, no React.

Step two

Press Alt+A

A toolbar appears. Every element you hover glows blue with its selector.

Step three

Click and type

Pick an element. Write what you want changed. Numbered markers mark the page.

Step four

Paste into Claude

Click Copy — structured markdown with selectors, positions, and context.

Output

Structured markdown an agent can act on.

Each annotation carries the data AI agents need to find the right code: selector path, CSS classes, bounding box, computed styles, nearby text.

Detail levels
Compact
Element name + your comment. One line each.
Standard
Adds selector path and selected text. Good default.
Detailed
Adds CSS classes, bounding box, nearby context.
Forensic
Full DOM path, computed styles, accessibility, URL, timestamp.
Keyboard
Alt+A Toggle annotation mode
⌘↵ Submit annotation
Esc Close or exit
